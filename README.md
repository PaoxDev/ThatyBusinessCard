# 🎨 Thaty Business Card
A modern digital business card designed to present personal information, social links, and professional identity in a clean and interactive way.

---

## 📖 What is this project
Thaty Business Card is a **frontend web application** that acts as a digital version of a traditional business card.
It allows users to:
* present personal and professional information
* share social media links
* create a modern and accessible online identity

Digital business cards are becoming increasingly popular because they allow instant sharing and a more interactive experience compared to traditional cards ([Spreadly][1]).

---

## 🎯 Purpose of the Project
* Build a **modern personal branding tool**
* Practice **UI design and layout structure**
* Create a reusable and scalable digital card
* Improve frontend development skills

---

## 🚀 Technologies Used
This project was developed using:
* **HTML5** → structure and semantic layout
* **CSS3** → styling, layout, and responsive design
* **JavaScript** → interactivity and dynamic behavior

---

## ⚙️ Features
* 👤 Personal profile section
* 🔗 Social media links
* 📱 Responsive design (mobile-friendly)
* 🎨 Clean and modern UI
* ⚡ Lightweight and fast (no backend required)

---

## 💡 How it works
This is a **static web application**, meaning:
* It runs entirely in the browser
* No server or database is required
* All content is displayed using HTML, styled with CSS, and enhanced with JavaScript

---

## 📦 Requirements
To run the project locally:
* A modern browser (Chrome, Edge, Firefox)

Optional:
* Local server (e.g. Live Server in VS Code)
